NiSqFile is an open-source file conversion and compression application built with Python Flask. Download the complete source code to run it locally or contribute to its development.

  • PDF processing utilities
  • Image conversion utilities

System Requirements:

  • Python 3.6 or higher
  • Flask framework
  • PyPDF2, PIL (Pillow), img2pdf libraries
  • python-docx for document processing
  • PyMuPDF (fitz) for PDF manipulation
